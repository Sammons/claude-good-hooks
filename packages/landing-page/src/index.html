Automate Your Claude Code Workflow

Simple CLI for applying automation hooks to enhance your Claude Code development workflow.

Get Started GitHub
$ npm install -g @sammons/claude-good-hooks
$ claude-good-hooks apply dirty --global
Hook applied successfully!

Key Features

Simple Setup

Install with npm and apply hooks instantly. No complex configuration needed.

Flexible Scoping

Apply hooks globally, per-project, or locally with full control.

NPM Integration

Install hooks from npm packages with a single command.

Quick Start

1

Install the CLI

npm install -g @sammons/claude-good-hooks
2

Install a Hook Package

npm install -g @sammons/dirty-good-claude-hook
3

Apply the Hook

claude-good-hooks apply dirty --global

Popular Examples

Git Status Hook

Automatically show git status to Claude before each response. 

claude-good-hooks apply dirty --global

Lint Before Edit

Run linting automatically before file modifications. 

const lintHook = {
  hooks: {
    PreToolUse: [{
      matcher: 'Write|Edit',
      hooks: [{ command: 'npm run lint' }]
    }]
  }
};

Session Context

Add git branch and recent commits to session context. 

const contextHook = {
  hooks: {
    SessionStart: [{
      hooks: [{
        command: 'git branch --show-current'
      }]
    }]
  }
};
